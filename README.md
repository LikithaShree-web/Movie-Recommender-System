# 🎬 Movie Recommender System

A content-based movie recommendation system built using Python, Machine Learning and Streamlit.

## Features

- Select a movie
- Get top 5 similar movie recommendations
- Fetch movie posters using TMDB API

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- TMDB API
- Hugging Face
